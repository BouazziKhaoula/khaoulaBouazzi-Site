# Dr. Khaoula Bouazzi — Bilingual Portfolio

Static bilingual portfolio in HTML/CSS/JavaScript. Compatible with GitHub Pages.

## Structure
- `index.html` — main one-page website
- `styles.css` — responsive design
- `script.js` — language switch FR/EN and mobile menu
- `assets/` — CV, logos and project visuals

## Local test
Open `index.html` in a browser.

## GitHub Pages
1. Create a public repository, for example `khaoula-portfolio`.
2. Upload all files and folders from this directory.
3. Go to `Settings > Pages`.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save and wait a few minutes.

## Google Analytics
After creating your GA4 Web Data Stream, paste the Google tag just after the `<head>` tag in `index.html`.

## Replace project visuals
Replace SVG files in `assets/projects/` with real screenshots using the same filenames, or update the image paths in `index.html`.
